# java-convenience-store-precourse

## 공통

* 잘못된 입력시 [ERROR] 예외
    * 그 부분부터 재입력
* 구매자의 할인혜택, 재고상황을 고려해 최종 결제 금액 계산
    * 상품별 금액 = 상품의 가격 * 수량 기반
    * 총구매액 = 총 상품별 금액 + 프로모션 및 멤버 할인 정책 반영
* 구매 내역, 산출 금액을 영수증으로 출력
* 영수증 출력 후 추가 구매 진행 가능

### 예외

* 상품과 수량 형식이 옳지 않은 경우
    * 올바르지 않은 형식으로 입력했습니다.
* 존재하지 않는 상품인 경우
    * 존재하지 않는 상품입니다. 다시 입력해 주세요.
* 재고 초과
    * 재고 수량을 초과하여 구매할 수 없습니다. 다시 입력해 주세요.
* 잘못된 입력
    * 잘못된 입력입니다. 다시 입력해주세요.

## 핵심 비즈니스

### 재고 관리

* 재고 수량으로 결제 가능 여부 확인
    * 재고 초과 예외
    * 해당 상품의 재고에서 차감

### 프로모션 할인

* 상품 당 프로모션 1개
* 프로모션 상품만 적용 가능
* 프로모션 기간 확인
* 프로모션 적용 상품보다 적게 가져오면 안내
* 프로모션 재고가 부족하면 일반 재고 사용
    * 일반 재고 사용 안내

### 멤버십 할인

* 프로모션 미적용 금액의 30% 할인
    * 프로모션 적용 후 멤버십 할인 적용
* 최대한도 8000원

### 영수증

* 구매 상품, 증정 상품, 금액 정보
* 구매한 순서대로 정렬해서 출력
